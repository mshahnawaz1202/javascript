## 🎯 Overview

This repository is a comprehensive collection of JavaScript learning materials, covering everything from fundamental concepts to advanced topics including Object-Oriented Programming, DOM manipulation, and asynchronous programming. Whether you're a beginner or looking to deepen your JavaScript knowledge, this repository provides structured examples and practical projects.

## 🚀 Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Node.js (optional, for running JS files outside browser)
- Code editor (VS Code recommended)

### Installation

```bash
# Clone the repository
git clone https://github.com/mshahnawaz1202/javascript.git

# Navigate to the directory
cd javascript

# Open in VS Code (optional)
code .
```

---

## 💻 How to Use

### Running HTML Files

```bash
# Simply open HTML files in your browser
open 06_DOM/one.html        # macOS
start 06_DOM/one.html       # Windows
```

### Running JavaScript Files

```bash
# Using Node.js
node 01_basics/01_variables.js

# Using VS Code with Code Runner extension
# Press Ctrl+Alt+N (Windows) or Cmd+Option+N (Mac)
```

### Browser Console

```javascript
// Open browser DevTools (F12)
// Paste code from any .js file into console
```
