1. onclick Event
   ================================================================================

Used to execute code when an element is clicked.

Example:

document.getElementById('owl').onclick = function(){
alert('owl clicked');
}

Limitation:

* Only one event handler can be attached.
* Less flexible than addEventListener().

---

2. addEventListener()

---

Modern and preferred way to attach events.

Syntax:

element.addEventListener(
eventName,
callbackFunction,
useCapture
);

Parameters:

* eventName      => click, mouseover, keydown, etc.
* callback       => function executed when event occurs
* useCapture     => false (default) or true

Example:

document.getElementById('owl').addEventListener(
'click',
function(){
alert('Owl clicked!');
}
);

Every event automatically provides an Event Object.

Example:

document.getElementById('owl').addEventListener(
'click',
function(e){
console.log(e);
}
);

Useful Properties:

e.type
e.target
e.currentTarget
e.timeStamp
e.clientX
e.clientY
e.defaultPrevented

4. Event Bubbling (Bottom-Up)

---

Default behavior in JavaScript.

Flow:

IMG → LI → UL → DIV → BODY

Code:

document.getElementById('images').addEventListener(
'click',
function(){
console.log('UL clicked');
},
false
);

document.getElementById('owl').addEventListener(
'click',
function(){
console.log('Owl clicked');
},
false
);

Output:

Owl clicked
UL clicked

Reason:
Event starts from child element and moves upward.

Occurs when third parameter is true.

Flow:

BODY → DIV → UL → LI → IMG

Code:

document.getElementById('images').addEventListener(
'click',
function(){
console.log('UL clicked');
},
true
);

document.getElementById('owl').addEventListener(
'click',
function(){
console.log('Owl clicked');
},
true
);

Output:

UL clicked
Owl clicked

Reason:
Event starts from parent and moves downward.

---

6. stopPropagation()

---

Stops event from moving further.

Code:

document.getElementById('owl').addEventListener(
'click',
function(e){
e.stopPropagation();
console.log('Owl clicked');
}
);

Result:

Only Owl clicked is printed.

Prevents browser's default behavior.

Example:

document.getElementById('google').addEventListener(
'click',
function(e){
e.preventDefault();
}
);

Result:

Google link will not open.

---

8. Event Delegation

---

Instead of attaching events to every image,
attach one event listener to the parent element.

Advantages:

* Better performance
* Less memory usage
* Handles dynamic elements

## Purpose:

When user clicks any image:

1. Detect clicked image
2. Find parent 
3. Remove that  from DOM
4. Image disappears

Code:

document.querySelector('#images').addEventListener(

```
'click',

function(e){

    // Print parent element of clicked item
    console.log(e.target.parentNode);

    // Print tag name of clicked element
    console.log(e.target.tagName);

    // Ensure only images are removed
    if(e.target.tagName === 'IMG'){

        // Print image id
        console.log(e.target.id);

        // Get parent LI
        let removeLi = e.target.parentNode;

        // Remove LI from DOM
        removeLi.remove();

        // Alternative Method:
        // removeLi.parentNode.removeChild(removeLi);

    }

},

false
```

);
